# Enterprise Sales & Marketing Analytics Dashboard


## 📋 Project Overview

This is a complete **interactive Power BI dashboard** built to analyze enterprise sales performance, product profitability, regional trends, customer behavior, and time-based growth.

The dashboard transforms raw sales data into clear, actionable insights for sales and marketing teams.

### 🎯 Business Problem Solved
Sales data lacked centralized visibility. Stakeholders needed better understanding of:
- Top performing products and categories
- Regional revenue and profit contribution
- High-value customers
- Month-over-month and year-over-year growth trends

### ✅ Key Features

- **Professional Star Schema Data Model** (Fact + Dimension tables)
- **6 Interactive Pages**:
  - KPI Overview with key business metrics
  - Product Performance (Top 5 by Net Sales & Profit, Category Share & Profitability)
  - Region & Market Analysis (Revenue & Profit by Region + Category contribution)
  - Customer Analysis (Sales by Customer, Average Order Value, Trend over Time)
  - Time Intelligence & Growth (Cumulative growth, YoY comparison, Peak analysis)
  - Detail Drill-through Page for focused analysis
- **Synchronized Slicers** (Region & Category) across all pages
- **Reset / Clear All Filters** button for smooth user experience
- Strong **Time Intelligence** using DAX measures
- Clean dark theme with consistent professional styling

### 🛠️ Technologies Used

- **Power BI Desktop**
- **DAX Measures** (KPIs, growth calculations, profitability %, cumulative revenue)
- **Star Schema Data Modeling**
- Import Mode

### 📊 Key Insights

- Electronics category drives **~88%** of total revenue
- Strong revenue momentum observed in late 2024
- Clear top-performing products and high-value customers identified

### How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Use Region and Category slicers
4. Click Reset button to clear filters
